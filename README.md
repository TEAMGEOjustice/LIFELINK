# 🩸 LifeLink — AI-Powered Real-Time Blood Donation Network

> From 2–6 hour delays to under 5 minutes.  
> LifeLink connects hospitals with the best available donors instantly using AI + real-time alerts.

---

## 🚨 Problem
Hospitals often rely on manual calls and fragmented networks to find blood donors. This leads to critical delays (2–6 hours) during emergencies.

---

## 💡 Solution
LifeLink is an AI-driven, real-time platform that:
- Matches the best donors based on compatibility, distance, availability, and trust score
- Sends instant alerts to multiple donors
- Confirms availability with a single tap
- Updates hospitals live

---

## 🔄 Core Flow
1. 🏥 Hospital creates an emergency  
2. 🤖 AI ranks top donors  
3. 🔔 Alerts sent in real-time  
4. 🙋 Donor accepts/rejects  
5. 📡 Hospital sees live status  

---

## ✨ Features
- 🤖 **AI Matching Engine** (FastAPI)
- 🔔 **Realtime Notifications** (Supabase Realtime)
- 🧑‍🤝‍🧑 **Donor Profiles & Availability**
- 🏥 **Hospital Dashboard**
- 🔒 **Supabase Auth + RLS (security-first)**

---

## 🛠 Tech Stack
- **Frontend:** Next.js 14, Tailwind CSS  
- **Backend:** Supabase (Auth, Postgres, Realtime, Edge Functions)  
- **AI Service:** Python FastAPI  
- **Maps/Distance:** Haversine / Google Maps (optional)
